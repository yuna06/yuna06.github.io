Pythonで正規分布について学ぼう！

Mon 25 March 2019 in 統計
#統計

なぜ正規分布が重要なのか？¶

正規分布という言葉を聞いたことのある人も多いかと思います。難しそうな言葉ですが、実はこの世界のあらゆる自然現象や社会現象は正規分布に従っていると言われています。

例えば、人の身長や一つの機械で作られた製品のサイズ、雨粒の大きさなど正規分布に従うものはたくさんあります。

　数式で正規分布を描く¶

正規分布の確率密度関数は下の通りです。正規分布に従うモデルにおいて、ある値xをとる確率はf(x)となります。 $$ f(x) = \frac{1}{\sqrt{2π}σ^2}\exp(-\frac{(x-μ)^2}{2σ^2}) $$

正規分布そのものを表す記号は下の通りです。

$$ N(μ, σ^2) $$

μは平均値、 σ^2は分散を表しています。

このように、正規分布は平均値と分散（標準偏差でも可）が決まれば、言い表すことができます。グラフで考えると平均値で山の横軸方向の位置が決まり、分散で山の広がりが決まります。

Pythonで正規分布を理解しよう。¶

In [13]:
import numpy as np
from scipy.stats import norm
import math
import matplotlib.pyplot as plt
In [21]:
x = np.arange(0,100,0.1)
a = norm.pdf(X,50,10)
b = norm.pdf(X,50,30)
c = norm.pdf(X,80,10)

plt.plot(x,a,color='red')
plt.plot(x,b,color='blue')
plt.plot(x,c,color='green')
plt.show()

上のようにscipyのnormというライブラリを使っていくつか正規分布の表を書いてみました。 赤いグラフの正規分布は平均値が50、分散が10です。青いグラフは分散が30と大きくなっているのでのっぺりした山になっています。緑のグラフは平均値が80なので右側に山が寄っています。

まとめ 正規分布に従うデータはたくさんあるので、気になったらグラフを書いてみましょう。
