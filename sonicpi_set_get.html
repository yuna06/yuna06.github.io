Rubyのメソッド？Sonic Piのメソッド

Sonic PiではRubyがプログラミング言語として利用されていますが、Sonic Pi特有の表現（関数定義法など）が多数登場します。
初めて見るメソッドが登場した時に、Rubyデフォルトのものであれば、Sonic Piのドキュメントを見ても説明されていませんし、Sonic Pi特有のものであれば、terminalなどで使うことができないことになります。
Rubyのメソッドか？Sonic Piのメソッドか？気にかけるようにしましょう。
今回紹介する、set, getはTime Stateを操る、Sonic Pi特有のメソッドです。それではいきましょう。

Time Stateとは？

Time Stateとは、Global memory store（プログラムの開始と同時に割り当てられ、終了するまで解放されることのないメモリ）のことです。setとgetで扱っていくことになります。

Set

Time Stateに持たせる情報は２つあります。

・持たせたい情報
・情報のキーネーム

書き方としては下のようになります。 
set :intensity, 3000
intensityがキーネーム、3000が情報になります。

Get

getを使って、setで設定した情報を取り出します。先ほど作った情報をキーネームで呼び出しましょう。 

get [:intensity]
これで大丈夫です。
