FoxDot Player Attribute

Published: Mon 25 March 2019
By yuna06

In foxdot.

tags: foxdot

SynthDefsは様々なオプションをとることができます。オプションは大きくattributeとeffectの２つに分けることができます。attributeは音階やタイミングを規定するもので、effetはどのような音を鳴らすかを決めるものです。
オプションのキーワードは次のコードを打ち込むことで確認することができます。 

print(Player.get_attributes()
>>
('degree', 'oct', 'dur', 'delay', 'blur', 'amplify', 'scale', 'bpm', 'sample', 'sus', 'fmod', 'pan', 'rate', 'amp', 'vib', 'vibdepth', 'slide', 'sus', 'slidedelay', 'slidefrom', 'bend', 'benddelay', 'coarse', 'striate', 'pshift', 'hpf', 'hpr', 'lpf', 'lpr', 'swell', 'bpf', 'bpr', 'bits', 'amp', 'crush', 'dist', 'chop', 'tremolo', 'echo', 'decay', 'spin', 'cut', 'room', 'mix', 'formant', 'shape')

attirbute

playerオブジェクトのattributeにダイレクトに値を代入することができるので、以下のコードは同じ意味になります。 

p1 >> pluck([0, 1, 2, 3], dur=1/2, sus=2)

p1 >> pluck()
p1.degree = [0, 1, 2, 3]
p1.dur = 1/2
p1.sus = 2

degree

ピッチを設定します。デフォルトスケールはC Majorにっています。 

p1 >> pluck([0, 1, 2, 3])

oct

octではノートを演奏したいオクターブを指定します。デフォルトでは5で、一オクターブが12半音なので、普通にplayerオブジェクトを再生すると60th半音で始まります。 

p1 >> pluck(oct=[4, 5, 6])

dur

音の長さを指定します。少数、分数も可能です。 

p1 >> pluck([0, 1, 2, 3], dur=[0.1,0.3,0.43,0.17])
0を使った場合演奏しなくなります。restを使うと、指定した時間ミュートします。 
p1 >> pluck([0, 1, 2, 3], dur=[1, 1, rest(2)])
